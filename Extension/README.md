## 📖 Overview

Quantum AI is an AI-powered Chrome Extension that helps users understand any webpage without leaving the browser. It extracts the main content from the current webpage and uses Google Gemini AI to provide instant summaries, explanations, context-aware answers, and voice assistance.

Whether you're reading documentation, GitHub repositories, blogs, research papers, Wikipedia articles, or news websites, Quantum AI acts as your personal AI assistant directly inside Chrome.

## 🚀 Installation

1. Clone this repository.

```bash
git clone https://github.com/your-username/Quantum-AI.git
```

2. Open **Google Chrome**.

3. Go to:

```text
chrome://extensions
```

4. Enable **Developer Mode**.

5. Click **Load Unpacked**.

6. Select the **extension** folder.

7. The Quantum AI extension is now installed and ready to use.

---

## 🎯 How It Works

1. Open any webpage.
2. Click the Quantum AI extension.
3. The extension extracts the important content from the page.
4. Content is sent to the AI backend.
5. Gemini AI generates summaries, explanations, or answers.
6. View the response or listen using voice output.
